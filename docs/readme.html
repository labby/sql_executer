Addon Help


SQL-Executer is a solution to maintain your database without any other tool.

Just crate your own sql statements end get them run.



Intention


Using dbWatchSite it is a duty, to clean the tables logfiles and lock requiries.
If you have to lock requires via dbWatchSite backend it may take a lot of time. So I created some queries in phpmyadmin and run them there though it might be fine to run them via backend of your installation.
This admintool is the solution.

Installation


The SQL-Executer is an Admin Tool.
Just install it like any other addon via LEPTON backend.

Usage


Please create your queries using phpmyadmin or something similar.
Now no select queries are possible. Keep in mind that you can't use the constant TABLE_PREFIX in SQL Executer.
Copy the query into the editarea and save.
Execute the query via icon.
